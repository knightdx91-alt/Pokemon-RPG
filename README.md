# Pokémon RPG

A browser-based Pokémon game (GitHub Pages) covering regions 1–7, played in 3D
with a 2D toggle, built to look and act **exactly** like Pokémon Platinum.

**Start here: [`GAME_PLAN.md`](./GAME_PLAN.md)** — the master vision and build
order. Sinnoh first; single-player to an exact-Platinum bar; multiplayer later.

Shared assets (3D map extraction, the Ultra Moon battle decomp, exact-UI
pipeline) come from the sibling `Pokemon-Game` repo.
